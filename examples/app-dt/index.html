Design Token Example

Although there was never an intention to (re-)introduce the mess we currently have in Chakra, it kind of came by itself... we (probably) need a way to change things:

Mode Speaks to which overall colorScheme to pick. Light, Dark.

Theme Speaks to which set of design-token-sets to override defaults with.
(We could of course ship just one theme (and no defaults to override)

ColorScheme Just changes two CSS keys:
'--klik-color-primary-interaction' and '--klik-color-primary-interaction-hover', with the chosen color (value 500 and 600 respecively)
(We could of course ship just one colorScheme, baked into the theme?)

so, yes - lets find a way to cook all into ONE CSS file, and expose as little as possible in :root
